Popular i18n libraries include [react-i18next](https://react.i18next.com/) and [react-intl](https://formatjs.io/docs/react-intl/). There are also framework specific libraries that often offer more support, such as [next-international](https://next-international.vercel.app/).

The following resources may help you get started.
* [Documentation](https://react.i18next.com/)
* [Comparison to others](https://www.i18next.com/overview/comparison-to-others)
* [Using text as keys](https://www.i18next.com/principles/fallback#key-fallback)
